### v2rx

#### you will learn

-  async page loading
-  code splitting
-  build & minify

####  how to start

```
npm install v2rx -g 
npm install
npm start or v2rx start

build dist:
v2rx build

```

just follow the instruction

you can config the app through v2rx.config.js

more details at [v2rx doc](https://stephenzhao.github.io/v2rx)
